◆ set()

bool Option_Float::set ( const std::string &  v)
virtual

Stores the given value after parsing it into a double.

The value is converted into a double and stored in "myValue". Then, "markSet" is called in order to know that a value has been set.

The method returns whether the value could be set (the return value from "markSet").

If the string could not be converted into a double, an InvalidArgument is thrown.
